# Playlist-Tracker
A lightweight and minimal browser extension (only for chrome rn) that adds checkboxes and a visual progress tracker to YouTube playlists, helping you track your learning or watching progress efficiently.

You Can:
 Add checkboxes to each video in a playlist
 Persistent progress using browser storage
 Automatically restores progress on reload
 Circular progress indicator (live %)

 How to use it:
 clone the repo:
```bash
git clone https://github.com/hangingpawn64/Playlist-Tracker.git
```
 Open: chrome://extensions/
 Enable Developer Mode
 Click Load unpacked
 Select the project folder
 Reload you Playlist tab.
